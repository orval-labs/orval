title: Installation
---

You can install Orval with <a href="https://npmjs.com" target="_blank">NPM</a> or <a href="https://yarnpkg.com" target="_blank">Yarn</a>.

Using NPM:

```bash
$ npm i orval -D
```

Using Yarn:

```
$ yarn add orval -D
```

Using pnpm:

```
$ pnpm add orval -D
```
