Orval 可以从任意有效的 OpenAPI v3 或 Swagger v2 规范生成**类型安全的 TypeScript 客户端**，支持 YAML 和 JSON 格式。

## 为什么选择 Orval？

团队通常把 OpenAPI/Swagger 当作前后端之间的接口契约。传统工具可以生成基础代码，但很难满足现代前端对类型、安全 mock 和数据获取集成的要求。Orval 的目标是把接口契约直接变成可维护的应用代码。

### 核心能力

- **生成 TypeScript 模型** - 为请求和响应提供完整类型约束
- **生成 HTTP 请求函数** - 生成可直接调用、类型正确的请求代码
- **生成 [MSW](https://mswjs.io/) mock** - 在没有真实后端时测试前端流程

## 支持的客户端

默认生成的客户端使用 [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)。你也可以根据项目需要配置 axios、React Query、Vue Query、SWR 等客户端。
