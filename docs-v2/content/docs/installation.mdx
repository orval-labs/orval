Install Orval as a dev dependency in your project.

## Package Managers

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab value="npm">
```bash
npm install orval -D
```
  </Tab>
  <Tab value="yarn">
```bash
yarn add orval -D
```
  </Tab>
  <Tab value="pnpm">
```bash
pnpm add orval -D
```
  </Tab>
  <Tab value="bun">
```bash
bun add orval -D
```
  </Tab>
</Tabs>

## Global Installation

You can also install Orval globally:

```bash
npm install orval -g
```

## Verify Installation

After installation, verify that Orval is installed correctly:

```bash
npx orval --version
```
